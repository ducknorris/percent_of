# PercentOf

Module that implements ```percent_of``` for Fixnum and Float classes in Ruby.

## How to use

Add it to your ```Gemfile```.

```
  # ...
  gem 'percent_of'
  # ...
```

You can use ```#percent_of``` by invoking:

```
  20.percent_of 100 # => "20.00%"
  24.44.percent_of 123.45 # => "19.80%"
```
